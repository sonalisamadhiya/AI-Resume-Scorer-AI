## 🚀 Key Features  

- 📄 Resume input and structured data handling  
- 🤖 AI-based resume scoring logic  
- 🔐 Secure user authentication (Login / Signup)  
- 💾 Persistent data storage using MongoDB  
- 🔄 REST API integration between frontend and backend  
- ⚡ Fast and responsive UI  
- 📊 Real-time feedback and scoring system  

---

## 🧠 How It Works  

1. User logs into the system  
2. Enters resume details (skills, experience, etc.)  
3. Backend processes the data  
4. Scoring logic evaluates the resume  
5. Result is displayed on the frontend  

---

## 🛠️ Tech Stack  

### 🔹 Frontend  
- React.js (Component-based UI)  
- JavaScript (ES6+)  
- Tailwind CSS / CSS  

### 🔹 Backend  
- Node.js  
- Express.js  

### 🔹 Database  
- MongoDB (MongoDB Atlas)  

### 🔹 Tools & Libraries  
- Axios (API calls)  
- Mongoose (ODM for MongoDB)  
- JWT (Authentication)  
- Nodemon (Development server)
